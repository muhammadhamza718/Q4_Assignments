# Car Rental Website

A modern, responsive car rental website built with HTML, CSS, and JavaScript, based on a Figma design.

## Getting Started

This is a static website. To view it:

1. **Open the website:**

   - Simply open `index.html` in your web browser
   - Or use a local development server like Live Server (VS Code extension)

2. **Using Live Server (Recommended):**

   ```bash
   # If you have Live Server installed in VS Code
   # Right-click on index.html and select "Open with Live Server"
   ```

3. **Using Python's built-in server:**

   ```bash
   # Python 3
   python -m http.server 8000

   # Then open http://localhost:8000 in your browser
   ```

## Features

- Responsive design for mobile, tablet, and desktop
- Interactive car search form with pick-up/drop-off details
- Car listing with like functionality
- Show more/less functionality for car listings
- Clean, modern UI based on Figma design

## Deployment

You can deploy this static website to any static hosting service:

- Netlify
- Vercel
- Firebase Hosting

Simply upload all project files to your chosen hosting service.
